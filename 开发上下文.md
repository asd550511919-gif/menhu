# 旅行社门户网站开发上下文

## 项目概述

这是一个旅行社门户网站项目，使用纯HTML/CSS/JavaScript开发，设计稿基于1920x1080分辨率，采用响应式缩放适配不同屏幕尺寸。

### 1. Mirada Travel 作品集合页 (mirada-gallery.html)

**功能描述：**
- 电影票图片轮播展示
- 显示3张图片：中间放大居中，上下两张在屏幕外只露出一小部分
- 点击箭头向下切换，有平滑过渡动画

**当前样式配置：**
```css
/* 上方卡片 */
.ticket-card.position-top {
    top: -250px;
    transform: translateX(calc(-50% + 450px)) rotate(2deg) scale(1.5);
    opacity: 0.4;
    z-index: 1;
}

/* 中间卡片（当前选中） */
.ticket-card.position-center {
    top: 50%;
    transform: translateX(calc(-50% + 450px)) translateY(-50%) rotate(0deg) scale(1.6);
    opacity: 1;
    z-index: 3;
    pointer-events: auto;
}

/* 下方卡片 */
.ticket-card.position-bottom {
    bottom: -250px;
    transform: translateX(calc(-50% + 450px)) rotate(-2deg) scale(1.5);
    opacity: 0.4;
    z-index: 1;
}
```

**关键特点：**
- 中间图片：scale(1.6)，方向为正(0deg)，完全可见
- 上下图片：scale(1.5)，有倾斜(±2deg)，大部分在屏幕外
- 向右偏移450px（因为电影票图片右侧有透明部分）
- 使用cubic-bezier缓动函数实现平滑过渡

**轮播逻辑：**
```javascript
let currentIndex = 0;
const totalTickets = 4;

function updatePositions() {
    const tickets = document.querySelectorAll('.ticket-card');
    tickets.forEach(function(ticket, i) {
        ticket.classList.remove('position-top', 'position-center', 'position-bottom', 'hidden');
        
        const prevIndex = (currentIndex - 1 + totalTickets) % totalTickets;
        const nextIndex = (currentIndex + 1) % totalTickets;
        
        if (i === currentIndex) {
            ticket.classList.add('position-center');
        } else if (i === prevIndex) {
            ticket.classList.add('position-top');
        } else if (i === nextIndex) {
            ticket.classList.add('position-bottom');
        } else {
            ticket.classList.add('hidden');
        }
    });
}

function nextTicket() {
    currentIndex = (currentIndex + 1) % totalTickets;
    updatePositions();
}
```

function resetPageScale() {
    const wrapper = document.querySelector('.frame-wrapper');
    if (wrapper) {
        wrapper.classList.remove('fade-out');

        wrapper.style.transform = wrapper.style.transform.replace('scale(1.1)', '');
    }
}

## 技术要点

### 1. 响应式缩放

所有页面使用统一的缩放逻辑：

```javascript
function scalePage() {
    const targetWidth = 1920;
    const targetHeight = 1080;
    
    const actualWidth = window.innerWidth;
    const actualHeight = window.innerHeight;
    
    const scaleX = actualWidth / targetWidth;
    const scaleY = actualHeight / targetHeight;
    const scale = Math.min(scaleX, scaleY);
    
    const wrapper = document.querySelector('.frame-wrapper');
    if (wrapper) {
        wrapper.style.transform = 'scale(' + scale + ')';
        wrapper.style.width = targetWidth + 'px';
        wrapper.style.height = targetHeight + 'px';
    }
}
```

### 2. 页面跳转动画

```javascript
function navigateToPage(pageName) {
    const wrapper = document.querySelector('.frame-wrapper');
    if (wrapper) {
        wrapper.classList.add('fade-out');
        setTimeout(function() {
            window.location.href = pageName + '.html';
        }, 600);
    }
}
```

```css
.frame-wrapper.fade-out {
    opacity: 0;
    transform: scale(1.1) !important;
}
```

### 3. CSS过渡动画

使用cubic-bezier缓动函数：

```css
transition: all 0.6s cubic-bezier(0.25, 0.46, 0.45, 0.94);
```

## 已解决的问题

1. **浏览器返回后页面放大**：通过visibilitychange、focus、pageshow事件重置页面缩放
2. **箭头图标阴影**：编辑SVG移除filter定义
3. **导航挂件不显示**：调整z-index到9999，确保在所有元素之上
4. **电影票轮播样式**：调整缩放、旋转、位置，实现从屏幕外滚入的效果

## 注意事项

1. 所有页面跳转使用`navigateToPage()`函数，确保动画效果
2. 导航挂件代码必须放在`</body>`之前
3. 电影票图片有透明部分，需要通过translateX调整位置
4. 使用`calc()`函数进行复杂的transform计算
5. z-index层级管理：背景 < 渐变 < 卡片 < 指示器 < 箭头 < 导航挂件
